# [DEC-007]: Remove Express Monolith

**Date:** 2026-05-29  
**Status:** `Accepted`

## Decision

Delete `server.ts` and run the app as a **Vite SPA** only. All former `/api/*` routes are replaced by Supabase Storage (client CRUD) and Edge Functions (`api`, `telegram-webhook`).

## Consequences

- `npm run dev` → `vite` (port 3000)
- `npm run build` → `vite build` only
- Telegram webhook URL: `{SUPABASE_URL}/functions/v1/telegram-webhook/{userId}`
